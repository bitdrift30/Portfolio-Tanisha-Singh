AskMyPDF (RAG System)

LLaMA 3 | Groq API | ChromaDB | Python

Built a Retrieval-Augmented Generation (RAG) system for answering questions based on PDF content. The solution uses SentenceTransformers to convert PDF chunks into dense vectors, which are then stored in ChromaDB, enabling the semantic retrieval of relevant document segments via vector similarity. For a given query, the top-matching document segments are retrieved and passed as context to LLaMA 3 (70B) via the Groq API to generate coherent, context-aware answers.

Legal Case Retrieval Tool

NLP | Vector Search | Cosine Similarity | Python

Developed an AI-driven tool to assist legal professionals in retrieving precedent case judgments based on textual case descriptions, thereby streamlining legal research workflows. This involved leveraging Natural Language Processing (NLP) to extract semantic meaning from case texts using the all-MiniLM-L6-v2 transformer model from Sentence Transformers and implementing cosine similarity for semantic search to efficiently rank and retrieve the most contextually relevant legal documents from a corpus.
